# REST Secrets API CRUD

A simple Node.js + Express application that demonstrates full CRUD operations using a REST API.

## 🚀 Features
- GET a secret by ID
- POST a new secret
- PUT (replace) a secret
- PATCH (update partially) a secret
- DELETE a secret
- Uses Axios for API requests
- Uses Bearer Token authentication

## 🛠️ Tech Stack
- Node.js
- Express.js
- Axios
- EJS
- REST API

## 📦 Installation

1. Clone the repository
2. Run:

```bash
npm install
npm start
```

3. Replace `YOUR_TOKEN_HERE` in `index.js` with your Bearer token.

Server runs on:
```
http://localhost:3000
```
